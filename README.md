## Run Locally

Clone the project

```bash
  git clone https://github.com/ihsansunman/realtime-chat-app
```

Go to the project directory

```bash
  cd ibb-data-dashboard
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
